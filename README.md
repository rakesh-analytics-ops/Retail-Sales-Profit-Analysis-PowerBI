# Retail Sales & Profit Analysis Dashboard

## Overview
This Power BI dashboard provides a comprehensive analysis of retail sales performance, profitability, customer segments, product categories, and regional trends.

## Dashboard Highlights

### Key Metrics
- Total Sales: 14.92M
- Total Profit: 1.52M
- Total Orders: 5K
- Shipping Cost: 108K

### Business Insights
- Top Region: West
- Top Category: Technology
- Top Product: Canon PC940 Copier
- Highest Performing Segment: Corporate

## Features
- Executive KPI Overview
- Regional Sales Analysis
- Top 10 Products by Sales
- Customer Segment Profitability
- Order Priority Analysis
- Product Category Distribution
- Interactive Filters
